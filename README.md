# Simple-landing-page-Assignment---SRV-Media

## Overview
This project is a responsive and accessible landing page developed for the Premier Schools Exhibition (PSE) assignment. It is built as per the provided Figma design and development guidelines using HTML, CSS, and vanilla JavaScript (no frameworks).

---

## Tech Stack
- HTML5 (Semantic)
- CSS3 (Custom CSS, BEM naming)
- Vanilla JavaScript

---

## Features
- Hero Section: Dual-axis slider with autoplay, pause on hover, swipe support, keyboard navigation, and ARIA roles
- Participating School Logos: Continuous CSS sling animation with alternating directions and pause on hover/focus
- Choose the School: Four-card layout on desktop, swipe-enabled slider with dots on mobile
- Exhibition Section: Full-section slider with consistent card height and optional autoplay

---

## Accessibility
- WCAG 2.2 AA focused implementation
- Semantic HTML and landmarks
- Skip-to-content link
- Keyboard navigation and screen reader support
- Supports prefers-reduced-motion

---

## Responsiveness & Testing
- Fully responsive on desktop, tablet, and mobile
- Tested on Chrome, Firefox, Edge, and mobile browsers
- HTML/CSS validated and accessibility checked using axe

---

## How to Run
Open index.html in any modern browser.
